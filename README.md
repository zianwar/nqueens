## NQueens
Solution for the [N-Queens puzzle](https://en.wikipedia.org/wiki/Eight_queens_puzzle) using recursive backtracking implemented in Go

### Usage

4x4 board example:

```bash
nqueens$ go build && ./nqueens 4

[. . Q .]
[Q . . .]
[. . . Q]
[. Q . .]

[. Q . .]
[. . . Q]
[Q . . .]
[. . Q .]
```
